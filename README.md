# SupermercadoApp - Backend (.NET 8)

## 📌 Objetivo
API REST para gestionar productos, categorías y reportes de precios.  
Parte del portafolio técnico de Felix.

## 🚀 Tecnologías
- .NET 8
- ASP.NET Core Web API
- SQL Server
- Entity Framework Core
- Swagger/OpenAPI
- GitHub para control de versiones
- Notion para documentación

## 📂 Estructura inicial
- Controllers → Controladores de la API
- Models → Entidades y DTOs
- Data → Contexto de base de datos
- Services → Lógica de negocio
- Migrations → Scripts de base de datos
